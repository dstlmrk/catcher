Formáty komunikace

Jediným podporovaným formátem je JSON. V dané hlavičce úvádějte MIME type application/json.

/createTournament

POST
POST ORGANIZER - Vytvoří nový turnaj.

/club/{id}

GET PUT DELETE
GET PUBLIC - Získání informací o konkrétním oddílu.
PUT ADMIN - Úprava oddílu.
DELETE ADMIN - Smazání oddílu.

/clubs

GET POST
GET PUBLIC - Získání kolekce všech oddílů.
POST ADMIN - Vytvoření nového oddílu.

/player/{id}

GET PUT DELETE
GET PUBLIC - Získání informací o konkrétním hráči.
PUT ADMIN - Úprava hráče.
DELETE ADMIN - Smazání hráče.

/players

GET POST
GET PUBLIC - Získání kolekce všech hráčů.
POST ADMIN - Vytvoření nového hráče.
