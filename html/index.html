Formáty komunikace

Jediným podporovaným formátem je JSON. V dané hlavičce úvádějte MIME type application/json.

/club/{id}

Oddíl

GET - Informace o konkrétním oddílu

public organizer admin

Požadavek
GET /api/club/2 HTTP/1.1
Host: catcher.zlutazimnice.cz
Content-Type: application/json
Odpověď
{
  "caldId": 8,
  "name": "PeaceEgg",
  "city": null,
  "country": null,
  "shortcut": "8XP",
  "user": {},
  "id": 2
}

PUT - Úprava oddílu

admin

Upravovat lze pouze sloupce shortcut, city a country (ISO 3166-1 alpha-3).

Požadavek
PUT /api/club/2 HTTP/1.1
Host: catcher.zlutazimnice.cz
Content-Type: application/json

{
  "city": "Písek",
  "country": "CZE",
  "shortcut": "PEA"
}
Odpověď
{
    "caldId": 8,
    "name": "PeaceEgg",
    "city": "Písek",
    "country": "CZE",
    "shortcut": "PEA",
    "user": null,
    "id": 2
}
